# Backend

## Commandes utiles
*Pour les commandes gradle, il faut être dans la racine du backend.*

**Supprimer le répertoire build**
```console
./gradlew clean
```
*peut-être utile lorsque les changements semblent pas être reconnus*

**Build l'application**
```console
./gradlew build
```

**Lancer l'application**
```console
./gradlew bootRun
```

**Lancer les tests**
```console
./gradlew test
```
